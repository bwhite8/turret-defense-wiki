# Modes & Maps

Different ways to play Turret Defense.

---

## Game Modes

### Regular
*Standard gameplay with all features enabled*

- All turrets available (based on research)
- All progression systems active
- Standard leaderboard
- Powerups enabled

**Best for:** Normal play, progression, leaderboard climbing

---

### Pulses Only
*Only Pulse Cannon turrets available*

| Rule | Effect |
|------|--------|
| Allowed Turrets | Pulse Cannon only |
| Other Features | Normal |

**Challenge:** Master the basics with only the starter turret. Tests positioning and upgrade efficiency.

**Tip:** Pack Tactics synergy (+8% damage per nearby Pulse) is crucial. Build tight formations.

---

### 3 Lives
*High stakes survival*

| Rule | Effect |
|------|--------|
| Starting Lives | 3 (instead of 20) |
| Quantum Core | Disabled |
| Other Features | Normal |

**Challenge:** Every enemy that reaches the core is devastating. Perfect defense required.

**Tip:** Prioritize crowd control (Cryo, Temporal). Never let enemies slip through.

---

### Sandbox
*Free experimentation*

| Rule | Effect |
|------|--------|
| Turret Cost | $0 for all unlocked turrets |
| Leaderboard | Disabled |
| Achievements | Disabled |
| Prestige Points | Not earned |
| Powerups | Disabled |

**Best for:** Testing strategies, experimenting with turret layouts, learning mechanics.

**Note:** No rewards or progression in this mode.

---

### HARDCORE
*Ultimate challenge*

| Rule | Effect |
|------|--------|
| Starting Lives | 1 |
| Quantum Core | Disabled |
| Permanent Upgrades | Disabled |
| Powerups | Disabled |
| All Turrets | Available (regardless of research) |
| Enemy Speed | +20% faster |
| Leaderboard | Separate hardcore leaderboard |

**Challenge:** One mistake ends your run. No bonuses, no safety nets.

**Tips:**
- Every turret must count
- Perfect positioning is essential
- Slow/freeze effects are crucial
- Know exactly when each enemy type unlocks

---

## Maps

### Classic
*The original battlefield*

| Property | Value |
|----------|-------|
| Size | 30 x 22 cells (1200 x 880 pixels) |
| Core Position | Center |
| Spawn Edges | All 4 sides |
| Scrolling | No |

**Tactics:**
- Enemies come from all directions
- Build outward from center
- Cover all approach angles
- Classic tower defense layout

---

### Large Square
*Massive arena for maze builders*

| Property | Value |
|----------|-------|
| Size | 120 x 120 cells (4800 x 4800 pixels) |
| Core Position | Center |
| Spawn Edges | All 4 sides |
| Scrolling | Yes (pan with arrow keys/WASD) |

**Tactics:**
- Enormous space for complex mazes
- Force enemies to walk long paths
- Strategic turret placement along routes
- Plan your maze before building

---

### Narrow
*Long corridor gauntlet*

| Property | Value |
|----------|-------|
| Size | 30 x 120 cells (1200 x 4800 pixels) |
| Core Position | Near bottom (5 rows from edge) |
| Spawn Edges | Top only |
| Scrolling | Yes (vertical) |

**Tactics:**
- Enemies only spawn from top
- Long path to core = many shots
- Perfect for piercing Rail Guns
- Build layered defenses down the corridor

---

## Mode + Map Combinations

Any mode can be played on any map!

### Recommended Combinations

| Combination | Experience |
|-------------|------------|
| Regular + Classic | Standard experience |
| Regular + Large Square | Epic maze building |
| Regular + Narrow | Rail Gun paradise |
| Pulses Only + Classic | True test of basics |
| 3 Lives + Narrow | Tense gauntlet run |
| HARDCORE + Classic | Ultimate challenge |
| Sandbox + Large Square | Experiment freely |

## Leaderboards

### Regular Leaderboard
- Tracks: Score, Waves, Kills
- All modes except Sandbox and Hardcore
- Combined across all maps

### Hardcore Leaderboard
- Separate from regular
- Only Hardcore mode runs
- Extra bragging rights

### No Leaderboard
- Sandbox mode
- For practice only
